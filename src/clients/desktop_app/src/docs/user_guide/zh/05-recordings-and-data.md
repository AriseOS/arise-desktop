## 4. 录制和数据管理

### 4.1 Recordings Library（录制库）

**Recordings Library** 会保留你历史上的录制：

- 每条录制会显示录制时间和部分元信息；
- 你可以重新打开录制，查看步骤或再次生成 Workflow。

### 4.2 录制详情

在录制详情页面，你可以：

- 查看录制过程中每一步的时间线；

- 删除不需要的步骤；
- 直接从该录制重新发起 Workflow 生成流程。

### 4.3 数据管理和 Collections

在 **Data** 区域，你可以管理 Workflow 使用的结构化数据：

- 每个 Collection 存储一类结果或参考数据；
- 打开 Collection 可以查看条目和字段结构。

未来版本可能会增加更多导入 / 导出能力，本章节主要说明当前界面的行为。
