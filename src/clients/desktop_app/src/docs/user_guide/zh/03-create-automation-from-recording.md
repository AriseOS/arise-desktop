## 3. 从录制创建自动化

本章节介绍从「完整做一次任务」到「生成可复用 Workflow」的主路径。

### 3.1 开始第一次录制

在 **首页** 中你可以：

- 点击 **Start Recording** 按钮开始录制；
- 如果你已经有 Workflow，可以在「Create new workflow」卡片中点击 **Start Recording**。

应用将会：

1. 打开或连接到浏览器；
2. 开始捕获你的操作。

### 3.2 完成这次任务

录制过程中：

- 像平时一样使用浏览器完成任务；
- 把需要重复的数据拷贝、输入步骤都包含进去。

录制时，Ami 通常会捕获：

- 页面跳转（打开页面、点击链接、切换标签页等）；
- 对按钮、链接等元素的点击操作；
- 在输入框和表单中的文字输入，包括粘贴操作；
- 你高亮选择的文本（例如用于复制或作为后续步骤的参考）；
- 关键的滚动行为和视图位置变化（帮助后续复现任务场景）。

这些信息会在后续生成 Workflow 时，用来理解你的意图，并尽量复现同样的操作路径。

完成后，通过录制控制按钮停止录制。

### 3.3 分析和清理录制

停止录制后，你会看到一个 **分析** 页面：

- 回顾录制过程中捕获的每一步；
- 可以删除不需要的步骤或调整说明文字。

### 3.4 生成 Workflow

在分析流程中，点击生成相关按钮即可开始 **生成 Workflow**：

- 应用会把录制转换为结构化的工作流；
- 你可以在保存前预览和调整。

后续章节会介绍如何管理和运行这些 Workflow。
