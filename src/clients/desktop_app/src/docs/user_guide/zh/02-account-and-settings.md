## 3. 账号与设置

### 3.1 注册与登录

- 在 **Register（注册）** 页面创建账号。
- 注册成功后会自动跳转到 **Login（登录）** 页面。
- 使用你的账号密码登录后，才能使用 Ami 的全部功能。

常见问题：

- 密码错误：应用会在登录页提示错误信息。
- 服务器不可用：可能看到连接错误，请检查后台服务是否已启动。

### 3.2 设置页面

从主界面进入 **Settings（设置）**，你可以：

- 查看基础账号信息（用户名、邮箱等）。
- 查看配额、试用信息和 Token 使用情况（如果已启用）。
- 退出当前账号。

当你退出登录时，Ami 会清理本地会话信息，并返回登录页面。
