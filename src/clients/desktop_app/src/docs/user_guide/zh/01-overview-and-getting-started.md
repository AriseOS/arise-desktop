# Ami 桌面客户端使用手册

## 1. 概览

本手册介绍如何使用 Ami 桌面客户端来录制你的日常操作，把它们变成可复用的自动化 Workflow，并运行这些自动化。

### 1.1 什么是 Ami 桌面客户端

- 通过学习你的浏览器操作，自动化重复性工作。
- 你只需要完整做一次任务，Ami 会把过程提炼成可反复使用的 Workflow。
- 所有操作都在桌面应用中完成，包括录制、生成和运行。

### 1.2 核心概念

- **Recording（录制）**：一次完整任务中，在浏览器里的操作轨迹。
- **Workflow（工作流）**：从录制或自然语言描述生成的自动化流程。
- **Execution（执行）**：某个 Workflow 的一次具体运行。
- **Collections / Data（集合 / 数据）**：Workflow 读写的结构化结果或知识。

## 2. 快速开始

### 2.1 安装和启动

- 按照安装向导安装 Ami 桌面客户端。
- 首次启动时，应用会自动检查运行环境（浏览器、后台服务等）。

### 2.2 首次启动流程

首次启动时，你通常会看到：

1. **Initializing / Loading** 页面：Ami 正在检查环境。
2. **Setup** 页面：如果需要安装或配置浏览器 / 后端服务。
3. **Login / Register** 页面：环境就绪后，登录或注册账号。
