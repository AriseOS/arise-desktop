# Ami Desktop User Guide

## 1. Overview

This guide explains how to use the Ami desktop app to record your work, turn it into reusable workflows, and run automations.

### 1.1 What is Ami Desktop

- Automates repetitive browser-based work by learning from your actions.
- Lets you record real tasks once and convert them into reusable workflows.
- Runs workflows and manages data directly from a desktop app.

### 1.2 Core Concepts

- **Recording**: A captured sequence of actions you perform in the browser.
- **Workflow**: A reusable automation generated from recordings or natural language.
- **Execution**: A single run of a workflow.
- **Collections / Data**: Structured results and knowledge that workflows read from and write to.

## 2. Getting Started

### 2.1 Install and Launch

- Install Ami Desktop following the system installer.
- Launch the app; it will perform a quick environment check when starting for the first time.

### 2.2 First Launch Flow

On first launch you will typically see:

1. **Initializing / Loading** screen while Ami checks your environment.
2. **Setup** screen if required tools (like the browser or backend) are not ready.
3. **Login or Register** screen after setup completes.

This chapter gives a high-level picture; later chapters explain each step in detail.
