# AgentCrafter 客户端

AgentCrafter 的用户交互客户端，提供网页和CLI两种方式与 AgentBuilder 系统进行交互。

## 目录结构

```
client/
├── web/                    # 网页客户端
│   ├── frontend/          # 前端界面
│   └── backend/           # Web API 服务端
└── cli/                   # 命令行客户端
    ├── src/              # CLI 源代码
    ├── config/           # CLI 配置文件
    └── docs/             # CLI 文档
```

## 交互方式

### 1. 网页客户端 (Web Client)
- 可视化界面，直观的Agent构建体验
- 实时预览和交互式配置
- 支持拖拽式工具选择和配置
- 内置代码编辑器和调试工具

### 2. 命令行客户端 (CLI Client)
- 快速的命令行交互
- 支持脚本化和自动化
- 适合开发者和高级用户
- 支持批量操作和管道操作

## 使用场景

- **快速原型**: 通过自然语言快速创建Agent原型
- **生产部署**: 完整的Agent开发、测试、部署流程
- **团队协作**: 多人协作的Agent开发和分享
- **模板管理**: Agent模板的创建、分享和复用
