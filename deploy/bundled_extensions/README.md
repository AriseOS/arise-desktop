# Bundled Chrome Extensions for Browser-Use

Pre-downloaded Chrome extensions for browser-use. Bundled to avoid downloading from Google servers (blocked in China).

## How It Works

1. Place `.crx` files here with the extension ID as filename (e.g., `cjpalhdlnbpafiamejdnhcphjbkeiagm.crx`)
2. On browser tool initialization, these files are copied to `~/.config/browseruse/extensions/`
3. browser-use automatically extracts them when it starts
